Run SZ RZ on Mac With iTerm2

Posted on 2017-11-26 | Edited on 2019-06-24 | In Shell | Comments:

sz rz 是通过 ZModem 协议在远程服务器和终端机间上传下载文件的利器。

安装 Homebrew

安装 lrzsz

配置 iTerm2

  1. 下载脚本文件，存放到 /usr/local/bin

  2. 设置 iTerm2 触发器
    打开 iTerm2->Preferences->Profiles->Default->Advanced->Triggers，单击 Edit 按钮，新建两条规则

至此，就可以正常使用 sz rz 去发送接收文件了。
