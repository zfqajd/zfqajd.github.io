邮箱配置

文件上传大小

上传超过100M的文件就会报 Server responded with 0 code. 错误。
官方给的解决方法：https://github.com/jumpserver/jumpserver/issues/270
使用 nginx 做反向代理，启动两个 jumpserver 进程，一个负责界面处理，一个负责上传(最大只能上传2G)。

录像回放黑屏

使用 ssh 登录跳板机，再连接服务器，录像回放是黑屏，从 nginx 的访问日志查看大小也是不对，应该是没有渲染。
临时解决方法是把 jlog_log 表中的 filename 字段设为 null，就可以正常播放了。
