Posted on 2018-01-24 | Edited on 2019-06-24 | In Shell

  • 题一
    使用 for 循环在 /oldboy 目录下批量创建 10 个 html 文件，名称格式为：随机的10个小写字母加固定字符串 oldboy 。

  • 题二
    将题一文件名中的 oldboy 全部改成 oldgirl(用 for 循环实现)，并且 html 改成大写。

  • 题三
    批量创建10个系统账号 oldboy01-oldboy10 并设置密码（密码为随机8位字符串）。

  • 题四
    写一个脚本，实现判断 192.168.10.0/24 网络里，当前在线用户的IP有哪些？

    脚本计算 1+2+3….+100=?

    批量添加20个用户，用户名为：user1-20，密码为 user 后面跟5个随机字符或数字。

    随机生成1到100的整数。判断输入的数是否是数字及在范围内，如果不是，提示重新输入；如果猜大了，提示输小的数字，猜小了，提示输大的数字。
