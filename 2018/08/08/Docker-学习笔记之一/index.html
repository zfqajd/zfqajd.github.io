安装 Docker CE

修改 Docker 数据存放目录

Docker 数据默认存放在 /var/lib/docker 目录，如果 /var 目录不是挂载独立分区且磁盘空间不够大，后期容易空间不足。
安装好 Docker CE，先不启动 docker 服务。

安装 Docker Compose

配置 Docker 加速器

国内网络环境去 Docker 官方拉取镜像非常慢，可配置 DaoCloud 加速器来提升速度。

  1. 注册 DaoCloud 帐号，登录生成配置命令。
  2. 在服务器运行配置命令，重启 docker 服务即可。
