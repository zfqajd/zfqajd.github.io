使用 XtraBackup 备份 Mysql 从库

Posted on 2019-03-06 Edited on 2019-06-24 In MySQL

数据库配置了主从同步，从库主要用来备份及数据查询。之前是通过mysqldump做逻辑备份，数据越大备份越耗时，磁盘的I/O也会在那个时段彪高。所以现在改用xtrabackup做物理备份。

从库每天0点创建完全备份，之后每小时创建增量备份。

从库环境：

  • 主机内网IP：192.168.1.101
  • 数据库数据目录：/data/mysql
  • 数据库备份目录：/backup/mysql/{FULL, inc1, inc2}
